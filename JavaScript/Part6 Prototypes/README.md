Part6 Prototypes — OOP & Prototypes in JavaScript

This folder contains detailed reference guides explaining Object-Oriented Programming (OOP) concepts in JavaScript and how prototypes work. Files included:

- `prototypes.md` — prototype chain, __proto__, Object.getPrototypeOf/setPrototypeOf, lookup rules.
- `constructor-functions.md` — classical constructor functions, `new`, `prototype` property, `new.target`.
- `classes-and-syntax.md` — ES6 `class` syntax, `extends`, `super`, `static`, private fields.
- `encapsulation-privacy.md` — private fields, closures, Symbols, WeakMap for private state.
- `polymorphism-and-inheritance.md` — overriding, `instanceof`, method resolution order.
- `prototype-methods-and-extensions.md` — adding methods to prototypes, `Array.prototype` example, caveats.
- `best-practices.md` — practical recommendations and resources.

Use these docs as both study notes and a reference when working with the example files in this folder.
